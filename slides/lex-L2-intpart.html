LEX Ideals, Integer Partitions, and $L_2(n)$


$\mathfrak P$ := Set of integer partitions with distinct parts

Proposition:

There are 1-1 correspondences: $$\mathfrak P \leftrightarrow \text{LEX Ideals of } k[x,y] \leftrightarrow \{L_2(n)\}_{n\geq 1}.$$

Example:

$$(5,4,2) \leftrightarrow (x^3,x^2y^2,xy^4,y^5) \leftrightarrow (1,2,3,3,2)$$
