Pure Modules

Pure $S$-module $M$

$M$ is pure if there is a degree sequence $${\bf d} = (d_0 < d_1 < \cdots < d_n)$$ such that $\beta_{i,j}(M)=0$ if $j \not = d_i$.

Example

$$\beta\left(\frac{\Bbbk[x,y]}{(x^2,y^2)}\right) = \begin{pmatrix} 1 & \text{.} &\text{.}\\ \text{.}& 2 & \text{.} \\ \text{.}&\text{.}& 1 \end{pmatrix}\ ,\ \ \ \ \ \ \ {\bf d} = (0,2,4)$$
